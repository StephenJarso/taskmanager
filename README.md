##  TaskManager
A terminal program that can:
   1. Add tasks
   2. List tasks
   3. Mark tasks as done
   4. Delete tasks
   5. Persist tasks in memory(for now)

   ## Maps Vs Slices

   When should i use a slice and When should i use a map?
   - Most beginners use slices
   - good developers choose intentionally

   ## maps
    - DO NOT preserve order
    - Do Not allow safe pointer access to values

    slice - ordered tasks
    Maps - Fast lookup
